# React skeleton (Vite + JSX)

This repository now contains a minimal React (JSX) skeleton using Vite.

Quick start:

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

Files added:

- `package.json` — project metadata and scripts
- `index.html` — Vite entry
- `src/main.jsx` — app entrypoint
- `src/App.jsx` — main component
- `src/index.css` — basic styles
- `.gitignore`

Let me know if you want TypeScript instead, a different bundler, or example routes/components.
